# Image Filtering (Manual)

## Objective
Manually implement:
- Histogram operations
- Image negation
- Blurring (mean, Gaussian)
- Edge detection (Sobel, Prewitt)
- Frequency domain filtering
- Thresholding
- Contrast adjustment

## Tools Used
- Python (Jupyter Notebook)
- NumPy
- Matplotlib

## How to Run
Open `03_image_filtering.ipynb` in Jupyter Notebook.
